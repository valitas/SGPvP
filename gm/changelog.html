SGPvP Change log

Version 35

Released 18 Jan 2014. Fixes engage action for non-premium accounts.

Version 34

Released 22nd Sept 2013. This only includes a minor fix to use the premium buttons when possible when selecting the number of rounds to attack. This works around a reported issue, where the script occasionally fails to select the rounds correctly and fires one round. That issue hasn't been reproducible in development environments, so it may be there still, but Premium users should not be affected by it now.

Version 33

Released 10th Sept 2013. This release doesn't add many visible new features, but there are major internal changes, hopefully for the last time.

  • The script runs now on the top Pardus frame, rather than the main ("nav") frame. This allows for catching key presses on all frames. More importantly, this means the script is injected once when you undock, and persists loaded for a long time, rather than being parsed and evaluated every time you nav. This is a big performance improvement.
  • New import/export keymap feature.
  • Better documentation.

Version 32

Released 2nd Sept 2013. This is a major update, featuring:

  • Configurable key map with pretty user interface — no mucking about in the source code.
  • New actions available: engage for fewer rounds, raid.
  • User interface code segregated from core and loaded on demand, for faster loading (core script size below 30K).
  • Primary development moved to Google Chrome, now fully functional, releases ported to Firefox from now on.
  • Verified working on Firefox 18 and later.

We started keeping a change log document here. If it's ever needed, earlier revision information is available to developers, from the git log.
