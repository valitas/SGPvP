SGPvP Change log

Version 43

Released 8 September 2017. Adds boost activations, level 7 armour, and minor fixes.

Version 42

Released 8 March 2017. New actions telerob and buy missile, plus bug fixes.

Version 41

Released 19 Oct 2016. A wide array of new actions, including switching combat modes, repair-or-attack keys, timebomb deployment and more. New download and update URLs for Firefox, due to Dropbox policy change. Source code now on Github.

Versions 38, 39, 40

Undocumented.

Version 37.1

Released 5 Jun 2015. Fixes bug introduced by Greasemonkey issue #2156 No changes to Chrome version.

Version 37

Released 2 Apr 2014. Changes disengage so that, if not blocked, only one server request is needed to jump to retreat tile. Fixes fast keypress choking for action "set ambush". Deals with disabled buttons (e.g. cloak when you have no AP) by navving.

Version 36

Released 1 Apr 2014. Fixes two issues: an apparent freeze when the user was hitting keys too fast, and retreat not working reliably under partial refresh.

Version 35

Released 18 Jan 2014. Fixes engage action for non-premium accounts.

Version 34

Released 22nd Sept 2013. This only includes a minor fix to use the premium buttons when possible when selecting the number of rounds to attack. This works around an issue where the script occasionally fails to select the rounds correctly and fires one round. That issue hasn't been reproducible in development environments, so it may be there still, but Premium users should not be affected by it now.

Version 33

Released 10th Sept 2013. This release doesn't add many visible new features, but there are major internal changes, hopefully for the last time.

  • The script runs now on the top Pardus frame, rather than the main ("nav") frame. This allows for catching key presses on all frames. More importantly, this means the script is injected once when you undock, and persists loaded for a long time, rather than being parsed and evaluated every time you nav. This is a big performance improvement.
  • New import/export keymap feature.
  • Better documentation.

Version 32

Released 2nd Sept 2013. This is a major update, featuring:

  • Configurable key map with pretty user interface — no mucking about in the source code.
  • New actions available: engage for fewer rounds, raid.
  • User interface code segregated from core and loaded on demand, for faster loading (core script size below 30 KB).
  • Verified working on Firefox 18 and later.

We started keeping a change log document here. If it's ever needed, earlier revision information is available to developers, from the git log.
